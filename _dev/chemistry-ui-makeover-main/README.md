# Breaking Bad Chemistry Lab

Интерактивная химическая раскраска, вдохновленная сериалом "Во все тяжкие".

## О проекте
Это веб-приложение позволяет изучать химию через раскрашивание различных объектов (например, пламя солей, носок с осадками, ракета) в правильные цвета, соответствующие химическим реакциям.

## Запуск локально
1. `npm install`
2. `npm run dev`

## Как добавить новый уровень
1. Добавьте файл данных уровня в папку `data/` (формат JS).
2. Загрузите изображение в `images/`.
3. Добавьте файл в `js/manifest.js`.

## Технологии
- Vite
- React
- TypeScript
- Tailwind CSS
- Shadcn UI
